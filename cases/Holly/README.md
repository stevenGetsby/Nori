# Holly Case

This is the case-centric artifact workspace for Holly Shit开心拉屎.

- `case.json`: machine-readable case manifest.
- `brief/original.md`: original user/business brief.
- `assets/raw/brand_materials/`: original brand and product assets.
- `assets/selected/`: curated assets selected for a specific workflow or public reference chain.
- `assets/public_refs/`: public reference images prepared for image generation providers.
- `runs/`: workflow runs, probes, smoke tests, and migrated legacy run outputs.
- `outputs/`: promoted outputs that are not tied to a single run layout.
- `showcase/`: curated human-facing export for demos and preview surfaces.

Machine indexes for this case are written to `data/artifact_index/`.
