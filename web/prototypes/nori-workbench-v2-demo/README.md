# Nori Workbench V2 Demo

This is the newer static Nori demo moved from the local `New project 2/`
scratch folder.

## Entry Files

- `demo.html`: primary Nori flow demo.
- `desktop_demo.html`: desktop-style variant.
- `pets_portal_miniapp_lofi.html`: separate miniapp-style lo-fi prototype kept
  with the same source bundle because it came from the same demo folder.
