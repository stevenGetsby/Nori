# Web Prototypes

This folder holds frontend prototypes and static UI experiments.

These files can be useful for product discussion, but they are not the shipped
Nori workbench. A prototype may use CDN scripts, bundled demo JS, fake data, or
static assets. Once a prototype starts calling real Nori APIs or becomes part of
the product surface, move it into a dedicated app folder under `web/`.
